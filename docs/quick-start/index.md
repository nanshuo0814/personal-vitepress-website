## 如何使用

###  1、浏览左侧侧边栏
你会看到清晰的目录分类标签，如 AI 资源导航、编程导航等等。你也可以直接使用左上角的搜索框直接搜索关键字即可，更加方便快捷。

### 2、选择感兴趣的分类
点击你想要探索的分类，进入相应的页面。

### 3、查看分类目录下的子类
在子类页面中，你将看到站长精心整理的相关站点列表或资源文档等等。

### 4、查看资源
现在找到了你心仪的资源，傻瓜式浏览查看，若资源为链接，点击链接即可快速跳转，开启你的探索寻宝之旅。
